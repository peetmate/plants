# Orchid Collection — Nairobi

A browsable, photo-illustrated catalogue of a private orchid collection grown in
Nairobi, Kenya. Each plant card shows the accepted name, photos, and cultural
notes (light, temperature, humidity, air movement, feeding, watering and media).

This is a **public, read-only** showcase built from a personal collection
database. Personal details (acquisition sources, prices, private notes and
precise garden locations) have been removed.

## Viewing

Open `index.html`, or visit the published site (GitHub Pages).

## Notes

- Single self-contained page; photos live in `Photos/`, keyed by plant UID.
- Cultural information is compiled from sources including the AOS, IOSPE, RHS
  and botanical gardens, and reflects one grower's conditions in Nairobi. Treat
  it as a guide, not a prescription.
